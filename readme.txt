This Plugin allows you to add your AddStars code snippet into the header of your website so that your reviews can show on Googles search results.

== Description ==

<b>Description</b>

The AddStars Review plugin can only be used with the AddStars code snippet generated from your Addstars Dashboard at [AddStars](https://addstars.com/ "AddStars") or [Addstars Dashboard](https://app.addstars.io/ "AddStars Dashboard")

This simple plugin lets you insert your code snippet into the <head> of your WordPress website with no need to edit or modify your theme files.

Features of the AddStar Review Plugin

	Very lightweight
	Quick to setup
	Simple to insert the AddStars script
	Insert code snippet into <head>
	Features “defer” in the script to load last
